自動化您的
廣告截圖任務

設定 CSS Selector，自動巡檢並截取指定廣告版位。支援 iPhone 14 Pro 模擬、長網頁截圖與操作錄影，讓報表製作更輕鬆。

下載 macOS 版 .dmg (Apple Silicon) 下載 Windows 版 .zip (Portable)

適用於 macOS 10.15+ 與 Windows 10/11

強大功能，
專為廣告營運設計

🎯

精準定位截圖

貼上網址並設定 CSS Selector，即可精準截取特定廣告版位，排除不相關內容干擾。

📱

行動裝置模擬

內建 iPhone 14 Pro 視窗模擬 (393×852)，真實呈現廣告在手機上的樣貌。

⚡️

自動化排程

設定定時任務（預設 60 分鐘），全天候自動巡檢並保存截圖證據。

🎥

完整記錄

支援長網頁捲動截圖 (Full Page) 與操作過程錄影 (MP4)，不錯過任何動態細節。

簡單三步驟
開始自動化

  • 01

    新增目標網站

    輸入網站名稱與網址，選擇是否需要錄影或長截圖。

  • 02

    設定 CSS Selector

    指定廣告版位的 CSS 選擇器（例如 .ad-banner），可設定多組。

  • 03

    啟動排程

    點擊立即執行測試效果，或開啟排程讓它自動在背景工作。
